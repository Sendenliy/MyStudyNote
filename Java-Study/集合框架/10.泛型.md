## 什么是泛型？

1. Java 泛型（generics）是 JDK 5 中引入的一个新特性, 泛型提供了编译时类型安全检测机制，该机制允许程序员在编译时检测到非法的类型。
2. 泛型的本质是`参数化类型`，也就是说所操作的数据类型被指定为一个参数
